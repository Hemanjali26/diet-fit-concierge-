[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let me walk you through our agent's internal workflow.
First, every request passes through our Security Checkpoint. If a threat is detected, it triggers the Security Event handler and terminates safely. If the check passes, the query goes directly to our Orchestrator Agent.
The Orchestrator delegates tasks using AgentTools: it asks the Diet Agent to formulate daily meals, and the Workout Agent to design fitness schedules.
These agents rely on our local Model Context Protocol Server side-panel to calculate exact macronutrient targets and metabolic statistics.
Finally, the flow enters the Human Approval node, where the user reviews the draft, allowing them to either request revisions or finalize the approved plan.
